# Loved Pets Animal Shelter

A modern, responsive pet adoption website built with React and Tailwind CSS, featuring a warm wooden theme aesthetic.

## Features

- **Home Page** — Welcome section with shelter philosophy, hours of operation, and contact information
- **Adopt Page** — Browse adoptable pets with photos, details, and status badges
- **Volunteer Page** — Information about volunteer opportunities and how to apply

## Tech Stack

- **React 19** — UI library
- **Vite** — Build tool and dev server
- **Tailwind CSS v4** — Utility-first CSS framework
- **Wouter** — Lightweight routing
- **Lucide React** — Icon library
- **Framer Motion** — Animations

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/loved-pets-shelter.git
   cd loved-pets-shelter
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

4. Open your browser to `http://localhost:5000`
